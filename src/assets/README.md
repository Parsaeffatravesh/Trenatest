# Assets

فونت‌ها، آیکن‌های SVG، تصاویر و استایل‌های سراسری (در صورت نیاز) در این پوشه نگهداری می‌شود.

## راهنما
- آیکن‌های خطی به‌صورت SVG قابل Tree-shake استفاده شود؛ ترجیحاً یک آیکن‌ست مشترک.
- فونت‌های فارسی/لاتین با `font-display: swap` و هاست محلی یا CDN مطمئن بارگذاری شوند.
- تصاویر بزرگ فشرده (WebP/AVIF) و با lazy-loading استفاده شوند.
- دارایی‌های تم‌خاص (مثلاً پس‌زمینه گلس‌مورفیسم) در زیرپوشه `themes/` قرار گیرند.
