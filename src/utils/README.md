# Utils

توابع کمکی بدون وابستگی UI در این پوشه قرار می‌گیرند.

## نمونه‌ها
- قالب‌بندی اعداد/ارقام (با پشتیبانی RTL و tabular).
- تبدیل واحد (Price Tick، حجم پایه/Quote)، محاسبه کارمزد.
- تابع throttle/debounce و رپر requestAnimationFrame برای فیدهای زنده.
- formatter تاریخ/زمان با time zone کاربر.

## اصول
- بدون وابستگی به DOM/React مگر در زیرپوشه جداگانه.
- تست واحد برای هر تابع الزامی است.
