# Theme and design tokens

این پوشه مرجع توکن‌های طراحی و راهنمای استفاده از تم دوگانه (تیره/روشن) است.

## فایل‌ها
- `tokens.css`: متغیرهای پایه رنگ، تایپو، فاصله، شعاع، سایه و توکن‌های مخصوص چارت/کامپوننت. تم پیش‌فرض تیره است و با `data-theme="light"` مقادیر تم روشن اعمال می‌شوند.

## نحوه استفاده
1. در ریشه اپ، `tokens.css` را ایمپورت کنید تا متغیرها در دسترس باشند.
2. روی تگ `<html>` یا `<body>` اتریبیوت `data-theme="dark"` یا `data-theme="light"` بگذارید.
3. برای اعداد مالی از کلاس `.mono` استفاده کنید تا فونت ثابت‌عرض اعمال شود.
4. کلاس `.sr-only` برای متن‌های کمکی دسترس‌پذیری (ARIA) آماده است.

## الزامات دسترس‌پذیری
- کنتراست متن روی `--color-surface` و `--color-bg` حداقل 4.5:1 بررسی شده است.
- `color-scheme` برای هر دو تم ست می‌شود تا مرورگر کنترل‌های پیش‌فرض را هماهنگ کند.
