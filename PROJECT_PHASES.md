# نقشه فازهای اجرای پروژه پنل معاملاتی

این سند فازهای اجرایی را به چهار بخش تقسیم می‌کند و معیار ورود/خروج هر فاز را مشخص می‌کند تا تحویل‌ها شفاف و قابل پیگیری باشند.

## فاز 1: بنیاد طراحی و معماری (تکمیل‌شده)
- **هدف**: تثبیت اهداف محصول، تعریف معماری فرانت‌اند و دارایی‌های پایه طراحی برای شروع اجرای UI.
- **دامنه اصلی**:
  - تأیید اهداف، شاخص‌های موفقیت و پرسوناهای هسته.
  - نقشه اطلاعات (IA) و فلوهای اصلی (جست‌وجوی نماد، تریدتیکت، پرتفوی، هشدار).
  - تصمیمات معماری UI (React + TypeScript، ساختار پوشه، الگوی ارتباط WebSocket/HTTP).
  - توکن‌های طراحی پایه (رنگ، تایپو، فاصله) با تم تیره/روشن.
- **خروجی‌های قابل تحویل**:
  - سند «فاز 1» با فلوها، تصمیمات معماری و توکن‌های CSS پایه.
  - چک‌لیست پذیرش شامل حالات خطا/شبکه و ریسپانسیو اولیه.
- **معیار خروج**:
  - فلوهای اصلی Annotated و تأیید شده.
  - توکن‌های دوتم با کنتراست ≥ 4.5:1 برای متن‌های اصلی.
  - الگوی معماری فایل/کامپوننت و استراتژی داده زنده تصویب شده.

## فاز 2: وایرفریم Hi-Fi و کیت کامپوننت (در حال اجرا)
- **هدف**: تولید وایرفریم‌های Annotated دسکتاپ/موبایل، تکمیل توکن‌ها و تعریف API رفتاری کامپوننت‌ها.
- **خروجی**: وایرفریم‌های نهایی، کیت کامپوننت با Props/States/Keyboard Interactions، گایدلاین چارت و ولیدیشن/امنیت.
- **معیار خروج**: پوشش RTL، Skeleton/Empty/Error در همه صفحات، تست کنتراست و تعامل کیبورد در Modal/Menu/Combobox.

## فاز 3: پیاده‌سازی اولیه و Storybook
- **هدف**: پیاده‌سازی اسکلت فرانت‌اند با Theme Switch، اتصال به فید نمونه، و ساخت Storybook/Design Tokens قابل مصرف توسعه.
- **خروجی**: ریپو کد با ساختار مصوب، Storybook با کامپوننت‌های کلیدی، چارت زنده اولیه، و اسکریپت‌های lint/test پایه.
- **معیار خروج**: TTI < 2s در دسکتاپ، FPS ≥ 50 در سناریوی 5k داده، چک‌لیست A11y حداقل 90% پوشش.

## فاز 4: سخت‌گیری کیفی، عملکرد و هندآف
- **هدف**: سخت‌گیری بر عملکرد، امنیت کلاینت، تست کاربردپذیری و آماده‌سازی هندآف نهایی به تیم اجرا/عملیات.
- **خروجی**: نتایج بنچمارک و تست کاربری، Hardened security checklist، مستند هندآف (Design Ops Page، بلیت‌های توسعه)، برنامه انتشار.
- **معیار خروج**: عبور از تست‌های عملکرد/امنیت، بازخورد کاربری با اقدامات اصلاحی بسته شده، چک‌لیست هندآف کامل.

## وضعیت فعلی
- **فاز فعال**: فاز 2 – وایرفریم Hi-Fi و کیت کامپوننت.
- **گام بعدی**: قفل Deliverable فاز 2 (مشخصات وایرفریم، کیت کامپوننت، A11y/i18n، تست کاربردپذیری) و آماده‌سازی ورودی فاز 3.
